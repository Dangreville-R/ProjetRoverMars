<---------------------Projet Rover Mars------------------>

Code complet permettant de choisir le mode automatique ou manuel pour le pilotage.

Le code comprends également l'affichage des distances sur l'écran oled lorsque le rover est en pilotage automatique via le capteur ultrason HC-SR04.

Il permet aussi l'affichage sur l'écran oled ( peu importe le mode de pilotage) de l'humidité, de la température et aussi du nombre de CO2 en ppm (parties par million) ( 1 ppm = 1 partie de substance pour 1 million de parties de mélange total.)
